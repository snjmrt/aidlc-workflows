# ASCII 図の基準

## 必須: 基本 ASCII のみ使用

**重要**: 互換性のため、図は必ず基本 ASCII 文字のみを使用する。

### ✅ 許可: `+` `-` `|` `^` `v` `<` `>` と英数字テキスト

### ❌ 禁止: Unicode の罫線文字

- 禁止: `┌` `─` `│` `└` `┐` `┘` `├` `┤` `┬` `┴` `┼` `▼` `▲` `►` `◄`
- 理由: フォント/プラットフォーム間で表示が不安定

## 標準 ASCII 図パターン

### 重要: 文字幅ルール

**箱の各行は（空白を含め）文字数が完全に同じでなければならない**

✅ 正しい例（全行 = 67 文字）:

```
+---------------------------------------------------------------+
|                      Component Name                           |
|  Description text here                                        |
+---------------------------------------------------------------+
```

❌ 間違い（幅が不一致）:

```
+---------------------------------------------------------------+
|                      Component Name                           |
|  Description text here                                   |
+---------------------------------------------------------------+
```

### 箱のパターン

```
+-----------------------------------------------------+
|                                                     |
|              Calculator Application                 |
|                                                     |
|  Provides basic arithmetic operations for users     |
|  through a web-based interface                      |
|                                                     |
+-----------------------------------------------------+
```

### 入れ子の箱

```
+-------------------------------------------------------+
|              Web Server (PHP Runtime)                 |
|  +-------------------------------------------------+  |
|  |  index.php (Monolithic Application)             |  |
|  |  +-------------------------------------------+  |  |
|  |  |  HTML Template (View Layer)               |  |  |
|  |  |  - Form rendering                         |  |  |
|  |  |  - Result display                         |  |  |
|  |  +-------------------------------------------+  |  |
|  +-------------------------------------------------+  |
+-------------------------------------------------------+
```

### 矢印と接続

```
+----------+
|  Source  |
+----------+
     |
     | HTTP POST
     v
+----------+
|  Target  |
+----------+
```

### 横方向フロー

```
+-------+     +-------+     +-------+
| Step1 | --> | Step2 | --> | Step3 |
+-------+     +-------+     +-------+
```

### ラベル付き縦方向フロー

```
User Action Flow:
    |
    v
+----------+
|  Input   |
+----------+
    |
    | validates
    v
+----------+
| Process  |
+----------+
    |
    | returns
    v
+----------+
|  Output  |
+----------+
```

## 検証

図を作成する前に:

- [ ] 基本 ASCII のみ: `+` `-` `|` `^` `v` `<` `>`
- [ ] Unicode 罫線なし
- [ ] 位置合わせはスペース（タブ不可）
- [ ] 角は `+` を使用
- [ ] **箱の全行が同一文字幅**（空白を含めて数える）
- [ ] テスト: 等幅フォントで角が縦に揃っていることを確認
